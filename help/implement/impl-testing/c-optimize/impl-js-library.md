# Javascript 程式庫檔案

JavaScript 程式庫檔案可在初次載入後，在使用者的瀏覽器中快取，以限制所需下載的資料量。

頁面特定變數應放置在 HTML 程式碼片段中。其他所有變數則應放在 JavaScript 程式庫檔案中。
