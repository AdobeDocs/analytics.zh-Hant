# 建立頁面載入規則的條件

建立可決定規則所觸發之頁面的規則。

1. 指定要在頁面的哪個位置觸發規則。

   當規則中有條件須依賴頁面內容時，規則在頁面上的觸發時間就變得更重要。

   ![](assets/conditions-page-load-rules1.png)

1. 指定導致規則觸發的條件。

   例如，您可以選取&#x200B;**[!UICONTROL 「路徑」]，以識別要為其觸發規則的特定頁面。**

   ![](assets/conditions-page-load-rules2.png)
