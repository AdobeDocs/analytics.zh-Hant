# 程式庫下載或 FTP 的測試規則

如果您使用程式庫下載或 FTP 傳送，或您沒有測試環境，可以使用重新寫入工具 (例如 Charles) 來測試未發佈的規則。此需求會依您的特定實施而有所差異。
