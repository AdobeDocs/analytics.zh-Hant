# 分類

對來源報告中的細分資料進行分組 (分類)，可建立「分類」。

例如，您可能會想分析顯示廣告，但這些廣告可能與電子郵件、合作夥伴、文字廣告與社交媒體促銷活動混雜在一起。您可以建立群組，如此即可個別加以分析。

如需更多詳細資料，請參閱:

* Analytics 說明與參考指南中的[分類](https://marketing.adobe.com/resources/help/en_US/reference/classifications.html)
