# 量度

量度是有關訪客活動的量化資訊，例如檢視、點進、重新載入、平均逗留時間、件數、訂購、收入等。

量度是報告的基礎，有助於您檢視和理解資料關係。量度可並排比較與您的網站相關的不同資料集。
