# 代碼管理器

代碼管理器可讓您下載網站和行動平台的資料收集代碼。

**[!UICONTROL 「分析&gt;管理&gt;代碼管理器]**」。

下載程式庫後，必須設定代碼以便傳送資料至正確的追蹤伺服器和報告套裝。如需其他實施資源，請參閱[開發人員和實施](https://marketing.adobe.com/resources/help/en_US/reference/developer.html)。
