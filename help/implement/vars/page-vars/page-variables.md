# 頁面變數

頁面變數會決定報表中維度和量度的值。

下列為實作中常用的變數：

* [`pageName`](pagename.md):頁面的名稱。
* [`campaign`](campaign.md):將此變數設為促銷活動追蹤的查詢字串參數。
* [`events`](events/events-overview.md):填入量度以用於報告。
* [`products`](products.md):如果您有電子商務網站，請在訪客檢視或購買產品時設定此變數。
