# dc

> [!IMPORTANT] 此變數已停用。 請改 `trackingServer` 用。

在舊版Adobe Analytics中，Adobe要求您指定要傳送資料至哪個資料中心。 將點擊傳送到錯誤的資料中心會導致資料遺失。

Adobe已透過允許任何實作傳送點擊至來改善此體驗 `sc.omtrdc.net`。 不再需要指定資料中心。
