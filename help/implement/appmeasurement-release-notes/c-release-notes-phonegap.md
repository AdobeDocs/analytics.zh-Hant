[iOS 文件](https://marketing.adobe.com/resources/help/en_US/sc/appmeasurement/ios/phonegap.html)。

[Android 文件](https://marketing.adobe.com/resources/help/en_US/sc/appmeasurement/android/phonegap.html)。
