## 未經測試的外掛程式 {#section_32BA7CAB37554A278170A728F1D65CE9}

下列外掛程式應可繼續運作，因為其基礎功能仍受支援，但外掛程式本身未經過測試而證明可相容。在移轉之前，您應先在開發環境中測試這些外掛程式。

* getActionDepth
* getCookiesAccepted
