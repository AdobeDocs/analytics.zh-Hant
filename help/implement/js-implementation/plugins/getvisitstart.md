# getVisitStart

決定是否開始新瀏覽。

**設定變數**

無

**參數**

c = (字串) 追蹤的 Cookie 名稱。

**傳回**

瀏覽的第一頁為 (整數) 1，其他為 0。

**範例呼叫**

```
s.eVar50 = s.getVisitStart("s_visit");
```
