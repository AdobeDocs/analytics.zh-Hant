# s.linkName

將顯示在自訂、下載或退出連結報表中的名稱。 超過 100 個字元即開始截斷。可設為任何字串。這是影像請求中的 `pev2` 參數。

If set with `linkType`, an image request will be sent as a download, custom or exit link.
