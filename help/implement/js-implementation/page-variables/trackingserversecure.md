# trackingServerSecure

 變數會用於第一方 Cookie 實施，以指定影像要求和 Cookie 寫入所在的網域。


<!--

 -->

用於安全頁面。若&#x200B;*`trackingServerSecure`* 未定義，則 SSL 資料會傳送至 *`trackingServer`*.
