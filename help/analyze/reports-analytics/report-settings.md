# 報表顯示設定和導覽

{{ra-eol}}

說明定義所有報表顯示方式的設定，以及將預設功能表選項對應至其簡化功能表中各個位置的資訊。

## 報表顯示設定和導覽 {#concept_09832A2CA0FF4982B1AA37C1B635220B}

**[!UICONTROL Analytics]** > **[!UICONTROL 元件]** > **[!UICONTROL 所有元件]** > **[!UICONTROL 報告設定]**

| 元素 | 說明 |
|--- |--- |
| **一般設定** |  |
| 包含相關度量區域 | 相關度量是與您目前檢視的報告密切相關的度量 (例如頁面檢視次數報告中的前 5 個頁面)。 |
| 在詳細資料區域中顯示 Internet 平均數 | 顯示取自數千個商業網站的指定統計數據的平均值。啟用後，該區域將顯示為報告摘要和報告詳情區域中的獨立欄。此功能僅用於技術群組中的流量報告，以及搜尋引擎、語言和網域報告。 |
| 顯示 Adobe 預設功能表結構 | 忽略「管理工具」(管理員在此處自訂報告功能表以符合使用者的使用偏好) 中的設定，將報告功能表還原為預設設定。 |
| 顯示報告時強制限定欄寬 | 強制限定報告欄寬，以便取得勻稱、美觀的效果。該設定在顯示三個以上度量時特別實用。 |
| **圖形和圖表** |  |
| 在趨勢圖中反白標示週末 | 垂直標示趨勢報告圖中的週末日期。確認週末後，評估「趨勢」報告將更加容易。 |
| 在圖形和摘要中包含預測 | 估計特定統計資料在未來的變化情況。啟用後，預測會出現在報告摘要區域。 |
| 在報告中包括「日曆事件」 | 追蹤與特定事件相關的網站績效。啟用後，這些事件將顯示在報告上。 |
| 使用 Flash 圖形 | 在報告中啟用 Flash 圖。Flash 圖為報告提供了更加直觀、更具互動性的影像，但您無法方便地複製或儲存該等影像。若要快速複製或儲存影像，請停用該選項 (影像將為 .gif 格式)。若您取消選取該選項，複製圖表按鈕不會出現在報告工具列中。 |
| 顯示已選定圖表中「所有其他」資料 | 在劃分到單一物件中排名最前 5 個頁面下方顯示所有其他資料。(橫條圖顯示您報告中排名最前面的 5 個網頁或其他資料。) |
| 在報表圖表中顯示「無」、「未指定」和「分類/建立書籤」資料。 | 顯示未收到指定度量的評分值的度量。 |
| 在排名報告上顯示走勢圖 | 在排名報告的總計欄位中顯示走勢圖。這可讓您快速檢視總體趨勢，無須另外產生報告。 |
| **加速** |  |
| 啟用報告加速器，更迅速地瀏覽報告 | 啟用報表加速器，此加速器會使用基於時間的演算法來快取處理最近請求的報表，並僅檢查發生頻率最高的特定項目，從而加快報表傳送作業。透過將請求的報告快取處理 15 分鐘，報告加速器可近乎瞬間抓取那些隨後請求的報告。當來回瀏覽、列印報告或頻繁存取相同報告時，此設定尤其管用。停用時，系統會在每次請求報告時重新產生報告。 |
| 啟用「控制面板加速器」並顯示可用快取版本 | 啟用控制面板加速器，它會儲存控制面板的快取版本，以加快後續檢視。透過將控制面板的視圖快取處理 24 小時，控制面板加速器可幾近瞬間抓取該視圖，因為繁瑣的資料庫查詢及處理會提前完成。若可用的快取版本超過 24 小時，將產生新的控制面板，並建立新的快取版本。同樣地，當您更新控制面板 (或控制面板上顯示的任意報告) 時，也會建立新的快取版本。快取是以使用者為基礎。檢視共用控制面板的其他使用者，依據自己對控制面板加速器的使用和控制面板的更新，將看到不同的版本。 |
| 啟用網路加速器提升報告效能 | 透過最佳化 Adobe 基礎結構和您所處環境之間的路徑，加快資料傳遞到您地點的速度。 |
| 啟用地區加速功能，在中國享有更快速的使用者體驗 | 「地區加速器」使用地區專屬加速網域，在特定地區提供更快的使用者體驗。目前地區加速功能僅支援中國地區。若為非中國地區的使用者啟用這項功能，會拖慢使用者體驗。啟用地區加速功能後，請務必重新登入，使設定生效。如果您要停用「地區加速器」，請取消選取此核取方塊。 |
| **語言/貨幣/編碼** |  |
| 千分位分隔符號 | 選取每千位使用一次的分隔符號 (小數點或逗號)。許多國家使用小數點來分隔千位數。(該分隔符號將套用於系統中的所有數字，而不僅僅是貨幣。) |
| 使用報表套裝的預設貨幣 | 指定是否要使用報表套裝的預設貨幣。 |
| 貨幣 | 您要用來換算資料的貨幣。在此設定中選擇值時，儲存在資料庫中的資料並不會受到影響，但是會顯示為根據當日貨幣匯率換算的值。當未設定貨幣選項時 (設定為預設)，不會換算貨幣換，所有值均以美元 (USD) 儲存和顯示。若要在處理資料時（顯示資料之前）轉換貨幣，請聯絡您的Adobe客戶團隊。 |
| 排程報表編碼 | Shift-JIS，用於日文字元編碼。EUC-JP (擴展 Unix 編碼)，主要用於日文、韓文和簡體中文。 |
| CSV 分隔字元 | 您要用來分隔 CSV 值的字元。 |
