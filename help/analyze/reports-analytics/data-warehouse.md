# Data Warehouse 請求

Data Warehouse 是指用於儲存和自訂報表的原始、未處理資料複本，可供您透過篩選資料的方式來執行。您可以要求報告依據您的特定問題，從原始資料顯示進階資料關係。與大多數報告是顯示在報告介面中的情況不同，Data Warehouse 報告是透過電子郵件或 FTP 來傳送，並且最多可能需要 72 小時的時間來處理。處理時間依查詢的複雜度以及請求的資料量而定。
