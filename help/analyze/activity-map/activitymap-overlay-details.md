# 覆蓋圖詳細資料

將滑鼠暫留在連結覆蓋圖上方時，會顯示覆蓋圖詳細資料。

覆蓋圖詳細資料會顯示為該連結追蹤的下列值：

* 量度
* 原始值
* 排名
* 百分比值
* 連結 ID
* 地區
* 顯示在頁面上的連結報表
