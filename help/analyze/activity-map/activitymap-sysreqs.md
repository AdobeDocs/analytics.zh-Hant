# 支援的系統和需求

下列網頁瀏覽器支援 Activity Map。

| 受支援的瀏覽器 | 版本 |
|--- |--- |
| Firefox | 最新版本 |
| Chrome | 最新版本.注意：Activity MapChrome外掛程式目前屬於「開發人員模式」外掛程式。 啟動此外掛程式時，您會收到一則訊息，鼓勵您停用開發人員模式擴充功能。 每次啟動外掛程式時，都會顯示此訊息。 |

>[!NOTE]
>
> 從連結收集的觀點來看，Activity Map 支援更多瀏覽器。

其他需求：

* AppMeasurement 代碼 1.6 版或更高版本。[更多...](/help/analyze/activity-map/activitymap-getting-started/activitymap-enable.md)
* Adobe Analytics標籤擴充功能v1.90或更新版本
* Web SDK 2.15.0版或更新版本。
* 必須啟用 Cookie 和 JavaScript。
