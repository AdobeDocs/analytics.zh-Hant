# Analysis Workspace 概述

Analysis Workspace可讓您快速建立分析並分享見解。 您可以使用拖放瀏覽器介面建立分析、新增視覺效果以生動呈現資料、組織資料集，以及與組織中的任何人共用及排程專案。

以下影片提供可能項目的簡短概述。

>[!VIDEO](https://video.tv.adobe.com/v/26266/?quality=12)

## Analysis Workspace關鍵領域

下圖和表格說明Analysis Workspace中的一些關鍵區域：

![Analysis Workspace 概觀](assets/analysis-workspace-overvew.png)

| 影像中的位置 | 名稱和函式 |
|---------|----------|
| A | **最左側邊欄：** 包含標籤，可將面板、視覺效果和元件新增至Analysis Workspace。 也包含用來開啟資料字典的資料字典圖示。 |
| B | **左側邊欄：** 此區域會根據左側邊欄中選取的標籤，包含個別的面板、視覺效果或元件。 |
| C | **面板：** 面板新增至Analysis Workspace時，會顯示於此處。 您可以將視覺效果和元件拖曳至您新增的任何面板。 |
| D | **報表套裝下拉式功能表：** 針對Analysis Workspace中的每個面板，報表套裝下拉式功能表可讓您選擇要作為資料來源的報表套裝。 |

## 登入 Adobe Analytics {#login}

若要開始使用 Analysis Workspace，請前往 [experience.adobe.com/analytics](https://experience.adobe.com/analytics) 登入 Adobe Analytics。預設會顯示「專案」頁面，除非您已選取特定專案。

## 建立 專案 {#new-project}

在Analysis Workspace中的分析稱為 [專案](/help/analyze/analysis-workspace/build-workspace-project/freeform-overview.md).

如需如何在Analysis Workspace中建立專案的相關資訊，請參閱 [建立專案](/help/analyze/analysis-workspace/build-workspace-project/create-projects.md).

## Analysis Workspace工程的構成要素 {#analysis}

您在Analysis Workspace中建立專案的基礎要素包括面板、表格、視覺效果和元件。

### 面板

**面板**&#x200B;可供在專案中組織可能包含許多表格和視覺效果的分析。Analysis Workspace 中提供的許多面板皆可根據用戶輸入的少量內容，產生完整的分析集。在左側邊欄中，選取頂端的&#x200B;**[!UICONTROL 「面板」]**&#x200B;圖示，即可檢視完整的可用[面板](https://experienceleague.adobe.com/docs/analytics/analyze/analysis-workspace/panels/panels.html)清單。

![](assets/build-panels.png)

### 視覺效果

**視覺效果**，例如長條圖或折線圖，可以透過視覺化方式生動呈現資料。 在左側邊欄中，選取中間的&#x200B;**[!UICONTROL 「視覺效果」]**&#x200B;圖示，即可檢視完整的可用[視覺效果](https://experienceleague.adobe.com/docs/analytics/analyze/analysis-workspace/visualizations/freeform-analysis-visualizations.html)清單。

![](assets/build-visualizations.png)

### 元件

[元件](/help/analyze/analysis-workspace/components/analysis-workspace-components.md) 在Analysis Workspace中包含下列項目：

* 維度

* 量度

* 區段

* 日期範圍

請參閱 [元件概觀](/help/analyze/analysis-workspace/components/analysis-workspace-components.md) 以深入了解這些元件類型。

這些元件類型都可新增至視覺效果（例如自由表格），以開始回答您的業務問題。

了解元件術語後，您就可以開始將元件拖曳至視覺效果（包括自由表格）中，以 [建立分析](https://experienceleague.adobe.com/docs/analytics/analyze/analysis-workspace/build-workspace-project/t-freeform-project.html?lang=zh-Hant).

![](assets/build-components.png)
