# 比較 Analysis Workspace 與 Ad Hoc Analysis

將臨機分析術語和任務與分析工作區進行比較。

分析工作區將臨機分析的大部分功能帶入瀏覽器工作流程。 雖然產品之間的某些術語和功能保持不變，但分析工作區中還有一些新術語和分析方法。

如需這兩個產品之間主要功能與系統需求的技術比較，請前往 [這裡](https://marketing.adobe.com/resources/help/en_US/reference/analytics-product-comparison.html)。

## 重要術語的比較 {#section_6109406B83B043A18E46D38F130B1D2E}

| Ad Hoc Analysis | Analysis Workspace |
|--- |--- |
| 專案 | 工作區或專案 |
| 工作區 | 面板 |
| 報表 | 自由表格 |
| 圖表／圖形 | 視覺化 |
| 階層：專案 > 工作區> 報表 | 階層：專案 > 面板 > 表格 |
| 排名、趨勢、總計報表範本 | 自由表格視覺化 |
| 流模板 | 流量視覺效果 |
| 流失 | 流失視覺效果 |
