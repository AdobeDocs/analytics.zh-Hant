# 關閉異常偵測

您可以在欄層級關閉異常偵測。

1. 在資料行中選取![設定](/help/assets/icons/Setting.svg)。
   ![](assets/turn-off-anomalies.png)

1. 取消選取&#x200B;**[!UICONTROL 顯示異常]**。
