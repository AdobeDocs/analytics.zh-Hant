# 項目圖表

可讓您查看您感興趣的值與其他效能範圍 (目標) 相比較或進行測量的結果。

![](assets/bullet-image.png)

項目圖表具有單一主要度量(例如目前的年繳型收入)，可讓您輸入定性範圍績效範圍(例如與目標收入比較)。您可以指定高、中和低。您可在「[!UICONTROL 視覺效果設定]」中指定目標範圍。
