# 散點圖

散點圖視覺化顯示維度項目與最多三個量度之間的關係。 它需要三個元件，並支援視覺化最多四個元件。

* 行元件（通常為尺寸）表示圖形上的每個點。 不同的列顯示為不同的彩色點。
* 最左側的欄（通常為量度）會繪出點在Y軸（垂直）上的位置。
* 第二欄繪製點在X軸（水準）上的位置。
* 第三欄會決定點的半徑。
* 散點圖視覺化會忽略自由表格中的所有後續欄。
