# 散點圖

顯示呈現的印象以及有多少獨特使用者看到那些印象。

每個資料點的大小提供視覺提示，代表觀看者接觸到廣告的平均次數。大小和資料會依據您選取的維度、日期範圍和篩選器而改變。

![](assets/scatter.png)

> [!NOTE]與散點圖關聯的表格至少需有兩個欄。第一欄定義 X 軸，第二欄定義 Y 軸。如果還有第三欄，散點圖會用來決定點的半徑。換句話說，欄 *1*、*2* 和 *3* 對應至 *X*、*Y* 和&#x200B;*點半徑*。
