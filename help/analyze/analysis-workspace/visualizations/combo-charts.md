# 組合圖可視化

>[!NOTE]
>
>此功能目前正在進行[有限測試](/help/release-notes/releases.md)。

組合圖使快速構建比較可視化變得容易，而無需先構建表。 您可以輕鬆地以行/欄組合方式查看資料，以便瞭解趨勢。

使用組合圖表

* 輕鬆將上週的訂單與上個月（和去年）的訂單進行比較 — 只需點幾下滑鼠即可完成。

* 在同一圖表上快速分析和比較多個指標（如唯一訪問者和收入）。

* 根據時間展望期內的函式（如累積平均值）分析度量。
