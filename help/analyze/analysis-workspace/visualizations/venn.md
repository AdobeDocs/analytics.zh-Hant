# Venn

此 [!UICONTROL Venn] 視覺效果可讓您拖曳最多3個區段(從 [!UICONTROL 元件])和一個量度來建立 [!UICONTROL Venn] 圖表。

![](assets/venn.png)

然後您可以在區段上暫留以取得更多百分比等的前瞻分析。

![](assets/venn_hover.png)

## 轉彎 [!UICONTROL Venn] 圖表 [!UICONTROL 自由格式] 表格

若要將 Venn 圖表轉換為自由表格，請按一下 **[!UICONTROL Venn]** 標頭旁的彩色點，並選取&#x200B;**[!UICONTROL 「管理資料來源]** > **[!UICONTROL 可用的資料來源]** > **[!UICONTROL 自由表格」]**。

## 標準化 [!UICONTROL Venn] 圖表

標準化 [!UICONTROL Venn] 圖表（移除其大小），前往其「視覺效果設定」圖示並選取 **[!UICONTROL 標準化]**.

![](assets/normalization.png)
