# 快速鍵（快速鍵）

Analysis Workspace地區提供快速鍵（鍵盤快速鍵），讓導覽、分析、新增面板和視覺化，以及民主化（共用）。

注意:
* `Ctrl` = Windows和 `Command` = Mac
* `Alt` = Windows和 `Option` = Mac

## 導覽動作

| 動作 | 快速鍵 |
| --- | --- |
| 建立新專案 | ctrl + shift + p |
| 開啟現有專案 | ctrl + o |
| 前往「面板」窗格 | alt + shift + 1 |
| 前往「視覺效果」窗格 | alt + shift + 2 |
| 前往「元件」窗格 | alt + shift + 3 |
| 摺疊/展開所有面板 | alt + m |
| 摺疊/展開作用中的面板 | alt + ctrl + m |
| 移至下一個面板 | alt + 向右鍵 |
| 移至上一個面板 | alt + 向左鍵 |
