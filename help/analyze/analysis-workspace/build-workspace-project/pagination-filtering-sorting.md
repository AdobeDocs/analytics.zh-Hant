# 分頁、篩選和排序表格

如需概覽，請在 YouTube 上觀賞此影片:

[分析工作區中的分頁、篩選和排序表格](https://www.youtube.com/watch?v=2zxpRPCGspg)

## Advanced filtering options {#section_36E92E31442B4EBCB052073590C1F025}

按一下篩選條件圖示，然後按一下自由表格資料表中維度旁的「顯示進階選項」，即可讓您使用下列標準進行篩選:

* 包含
* 不包含
* 包含所有字詞
* 包含任何字詞
* 包含片語
* 不包含任何字詞
* 不包含此片語
* 等於
* 不等於
* 開始於
* 終止於
