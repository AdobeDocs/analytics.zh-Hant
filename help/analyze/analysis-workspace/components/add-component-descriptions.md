# 新增元件說明

說明可協助使用者在Analysis Workspace中建置專案或檢視報表時，最善用各種元件。

Analytics 管理員可以新增元件 (維度、量度、區段和日期範圍) 的說明<!-- either within the Report Suite or using the Data Dictionary directly within Analysis Workspace-->。
