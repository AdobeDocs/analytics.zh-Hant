# 管理注釋

的 [!UICONTROL 元件] > [!UICONTROL 注釋] manager提供了多種管理注釋的方法，如共用、篩選、標籤、批准、複製、刪除和標籤為收藏夾。

的 [!UICONTROL 注釋] manager向您顯示您擁有的所有注釋，這些注釋已作用於所有項目，並且已與您共用。

>[!NOTE]
>
>[!UICONTROL 注釋] 為特定項目建立的項目不顯示在管理器中。
