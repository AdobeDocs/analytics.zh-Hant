# 時間分段維度

「時間分段」功能會擷取所收集的點擊的時間戳記，並採用更有意義的維度加以分段，例如「小時」或「星期」。

時間分割維度是根據報表套裝或虛擬報表套裝的時區。 這類維度是由 Analysis Workspace 所提供，並且可以協助回答以下問題：

* 以長久的日期範圍來看，一天之中何時是訪客最常存取我的網站或應用程式的熱門時段？
* 在我的網站或應用程式中，是「星期」還是「小時」的轉換率較高？
* 我的週末銷售額與工作日銷售額有何不同？
* 某個行銷促銷活動在上午或下午產生較高的轉換率？

>[!NOTE]時間分段維度僅可在 Analysis Workspace 中使用。若要在其他Analytics解決方案中使用時間分割維度，您可以實 [作getTimeParting外掛程式](https://marketing.adobe.com/resources/help/zh_TW/sc/implement/getTimeParting.html)。

Analysis Workspace 中的時間分段維度包括：

| 維度 | 範例值 |
|--- |--- |
| 小時 | 0-23 |
| 上午/下午 | 上午，下午 |
| 星期 | 星期一、星期二、星期三、星期四、星期五、星期六、星期日 |
| 週末／工作日 | 週末、工作日 |
| 每個月的第幾天 | 1-31 |
| 月份 | 1-12月份 |
| 一年中的第幾天 | 1-366 |
| 季別 | 第1、2、3、4季度 |
