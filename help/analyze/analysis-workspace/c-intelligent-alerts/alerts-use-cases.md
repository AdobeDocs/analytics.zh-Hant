## 從表格選取範圍建立警報 {#section_AE6D42E1255D498D908A2FA60370A419}

在自由表格中，此時可以用滑鼠右鍵按一下表格列並選取&#x200B;**[!UICONTROL 「從選取範圍建立警報」]**，建立內容感知警報。

![](assets/alert_selection.png)

如此會立即預先填入警報產生器，使用正確的量度和篩選器建立警報：

![](assets/prepopulated_alert.png)
