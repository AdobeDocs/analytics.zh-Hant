# 固定日期和儲存的日期範圍

{{legacy-arb}}

當您指定固定日期或日期範圍時，Report Builder會顯示報表行事曆，讓您選取日期、日期範圍或預設日期。 您也可以儲存固定日期作為範本，並指定儲存的日期適用於目前的報表套裝還是適用於所有報表套裝。

1. 在「請求精靈: 步驟 1」中選擇&#x200B;**[!UICONTROL 「固定日期」]**。
1. 按一下日期連結。
1. 選擇日期並按一下&#x200B;**「確定」**。
1. 若要儲存日期範圍，請按一下&#x200B;**[!UICONTROL 「儲存日期」]**。
