# 日期縮寫

{{legacy-arb}}

時間縮寫是指期間開始時的特定日期，而非天數範圍。 天數範圍是透過指定兩個時間縮寫或辭彙的開始和結束日期來表示。

如需使用正確語法的詳細資訊，請參閱[語法備註](/help/analyze/legacy-report-builder/data-requests/configuring-report-dates/c-customized-date-expressions/examples-of-date-ranges-using-customized-expressions.md#section_555D6563B2D94FA3BDD801DC0B8C289D)。

* cd =目前日期（今天）
* cw =本週（本週的第一天）
* cm =當月（當月的第一天）
* cq =目前季度（目前季度的第一天）
* cy =當年（當年的第一天）

在以下清單中，*[UNIT]* 表示任何整數乘數。

* [UNIT] d = 天
* [UNIT] w = 週
* [UNIT] m = 月
* [UNIT] q = 季
* [UNIT] y = 年
