# 日期縮寫

時間縮寫表示期間開始的特定日期，而不是某個範圍內的日期。天數範圍是透過指定兩個時間縮寫或辭彙的開始和結束日期來表示。

如需使用適當語法的相關資訊，請參閱[語法附註](/help/analyze/legacy-report-builder/data-requests/configuring-report-dates/c-customized-date-expressions/examples-of-date-ranges-using-customized-expressions.md#section_555D6563B2D94FA3BDD801DC0B8C289D)。

* cd = 當天 (今天)
* cw = 當週 (當週的第一天)
* cm = 當月 (當月的第一天)
* cq = 當季 (當季的第一天)
* cy = 當年 (當年的第一天)

在以下清單中，*[UNIT]* 表示任何整數乘數。

* [UNIT] d = 天
* [UNIT] w = 週
* [UNIT] m = 月
* [UNIT] q = 季
* [UNIT] y = 年
