# 使用自訂運算式的日期範圍範例

{{legacy-arb}}

提供在自訂運算式中使用日期範圍的範例、附註及語法附註。

此表格假設今天的日期是2011年11月10日星期一（使用西曆）。

| 範例 | 日期範圍 | 自訂運算式 | 報告的日期範圍 |
|---|---|---|---|
|  | | **從** | **結束日期** |
| 1 | 兩週前 | `cw-2w  \| cw-1w-1d` | 10月26日至11月1日 |
| 2 | 去年第五個月的前3天 | `cy-1y+4m  \| cy-1y+4m+2d` | 2010年5月1日至5月3日 |
| 3 | 一整週，4週前開始 | `cw-4w  \| cw-3w-1d` | 10月12日至10月18日 |
| 4 | 前一年中的最後一週 | `cw-53w  \| cw-52w-1d` | 2010年11月至11月9日 |
| 5 | 2個月前開始的一個月 | `cm-2m  \| cm-1m-1d` | 9月1日至9月30日 |
| 6 | 12個月前的前一年 | `cm-12m  \| cm-11m-1d` | 2010年11月1日至11月30日 |

## 範例附註 {#section_37801B0D6D364ABAA8DCE3A4C0123B2C}

**範例 1**

如果今天是2011年11月10日星期一，則以目前日期減去一週即可取得10月的最後一整週。

**範例 2**

將四個月加到年初（1月），以取得5月；將兩天加到每月的第一天，以取得該月的第三天。

## 語法附註 {#section_555D6563B2D94FA3BDD801DC0B8C289D}

涵蓋大多數日期範圍的自訂運算式，可以透過將兩個辭彙與運運算元連結來建立。 辭彙是整數乘數和句點縮寫的組合。 18d是辭彙的一個範例。 運運算元的範例為+。

* 運運算元和字詞之間不允許使用空格。
* 僅限使用以下縮寫：cd cw cm cq cy d w m q y
* 最佳做法是在開始日期和結束日期中使用相同的日期參考：cd、cd 或 cw、cw 或 cy、cy。 混合日期參考可能會導致一年中某些時間的日期無效。
* d w m q y縮寫的有效倍數是透過縮寫前面的整數( 1 2 3 ... )形成的，例如53d 3w 5q 9m 2y
* 不允許非整數。
* 請勿在縮寫前面只加零。 例如，不允許0w。
* 以下運算子可用來連接縮寫：+ -
* 由於日期範圍必須相對於目前期間進行計算，因此運算式中的第一個詞語一律以c開頭。
