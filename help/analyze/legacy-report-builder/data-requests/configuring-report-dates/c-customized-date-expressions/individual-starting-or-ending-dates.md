# 個別的開始日期或結束日期

{{legacy-arb}}

開始日期和結束日期的縮寫。

* cd-1d =昨天
* cm-1d =上個月的最後一天
* cw-1w =上週的第一天
* cw+1w-1d =本週的最後一天
* cd-2d =兩天前
* cw-2w =一週的第一天，兩週前

>[!NOTE]
>
>如果您將日期設定為未來的日期，則不會傳回值。
