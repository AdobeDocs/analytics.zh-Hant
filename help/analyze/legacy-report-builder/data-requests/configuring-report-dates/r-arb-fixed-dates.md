# 請求精靈定義 - 固定日期

{{legacy-arb}}

Report Builder 中固定日期的定義。

| 欄位 | 定義 |
|--- |--- |
| 此報表套裝 | 使儲存的日期範圍僅供目前報表套裝之用。 |
| 所有報表套裝 | 使儲存的日期範圍供登入使用者可存取的所有報表套裝使用。 |
| 輸入參數名稱 | 輸入儲存日期的名稱。您可以將此日期用作Report Builder中的範本。 在任何可以套用儲存日期的位置就會出現此名稱。變更範本名稱不會影響先前建立之資料請求所用的日期組態。 |
