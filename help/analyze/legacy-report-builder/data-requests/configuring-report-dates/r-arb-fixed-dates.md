# 請求精靈定義 - 固定日期

{{legacy-arb}}

Report Builder 中固定日期的定義。

| 欄位 | 定義 |
|--- |--- |
| 此報表套裝 | 讓儲存的日期範圍僅適用於目前的報表套裝。 |
| 所有報表套裝 | 讓登入的使用者可存取所有報表套裝的已儲存日期範圍。 |
| 輸入引數名稱 | 輸入儲存日期的名稱。 您可以將此日期當做Report Builder中的範本。 此名稱會顯示在您可以套用已儲存日期的任何位置。 變更範本名稱不會影響先前建立資料請求中使用的日期設定。 |
