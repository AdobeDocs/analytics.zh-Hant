# 排程已啟用巨集的活頁簿

{{legacy-arb}}

Adobe Report Builder 支援 .xlsm 格式，可讓您排程已啟用巨集的活頁簿。

如果您需要安全地排程、處理及接收已啟用巨集的活頁簿，這個功能會很有用。

>[!IMPORTANT]
>
>雖然 Report Builder 可讓您排程含有巨集的活頁簿，但每次排程運作時，這些巨集都不會執行。 只有在活頁簿在原生Microsoft Excel應用程式中開啟時，才能使用這兩項功能。

含有巨集的排程活頁簿只能以啟用巨集的格式(.xlsm)傳送，因為所有其他支援的格式（xls、xlsx、pdf、word、csv或txt）都會從活頁簿中移除巨集。
