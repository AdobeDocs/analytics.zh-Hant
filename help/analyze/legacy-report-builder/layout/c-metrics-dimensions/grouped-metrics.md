# 將流量與商務量度合併為同一請求

{{legacy-arb}}

您可以將流量與商務量度型別結合在一個請求中，而非建立個別請求。

流量和商務量度會顯示在「請求精靈：步驟2」的相同量度類別中。 群組類別標示為&#x200B;**[!UICONTROL 標準]**。
