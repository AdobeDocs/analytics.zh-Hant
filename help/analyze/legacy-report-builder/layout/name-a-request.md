# 為請求命名

{{legacy-arb}}

您可以為請求指定自訂名稱，該名稱會顯示在「請求管理員」中。

如需詳細資訊，請參閱[格式顯示標題](/help/analyze/legacy-report-builder/layout/t-format-display-headers.md)。
