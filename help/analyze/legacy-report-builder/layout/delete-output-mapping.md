# 刪除輸出對映

{{legacy-arb}}

在編輯活頁簿時，如果您想要騰出一些空間給新請求使用，便需要刪除請求。

釋放空間的唯一方式是刪除對應至儲存格的請求。 如果您藉由移除列或欄來刪除試算表中的請求，則基礎請求的有效性將會遺失（且在重新整理時將會失敗）。 Excel功能表選項&#x200B;**[!UICONTROL 編輯]** > **[!UICONTROL 清除內容]**&#x200B;會移除儲存格中顯示的值。 重新整理儲存格的內容即可還原內容。

若要移除試算表儲存格的特定對應，請找出列、欄或量度專案，然後按一下&#x200B;**[!UICONTROL 刪除]**。
