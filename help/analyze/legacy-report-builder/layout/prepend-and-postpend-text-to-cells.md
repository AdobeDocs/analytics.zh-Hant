# 在儲存格的開頭或結尾附加文字

{{legacy-arb}}

對於「樞紐配置」或「自訂配置」請求中的數值或字串值，您可以在儲存格的開頭或結尾附加字串或字元。

此選項可在[!UICONTROL 請求精靈：步驟2]視窗中使用。

**附加文字於開頭：**&#x200B;在儲存格中加入數值或字串值形式的前置詞。

**附加文字於結尾：**&#x200B;在儲存格中附加數值或字串值。
