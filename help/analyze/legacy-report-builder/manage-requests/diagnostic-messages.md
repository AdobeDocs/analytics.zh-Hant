# 診斷訊息

{{legacy-arb}}

當您重新整理請求時，診斷訊息可提供請求的狀態資訊。

![綠色核取記號圖示，表示要求已成功重新整理。](assets/icon_notice_success.gif) 表示自開啟此活頁簿後，要求已成功重新整理。 診斷欄提供有關請求的其他資訊，包括請求使用的時間、對應請求的時間，以及請求是否使用先前快取的資訊。

![紅色三角形的圖示，帶有驚歎號，表示要求重新整理失敗。](assets/icon_notice_warn.gif) 當請求無法成功重新整理時，會顯示此警報。 診斷欄中會指出錯誤，指出失敗的可能原因。 請檢視[疑難排解區段](/help/analyze/legacy-report-builder/troubleshoot.md)，以取得所選失敗訊息的說明。
