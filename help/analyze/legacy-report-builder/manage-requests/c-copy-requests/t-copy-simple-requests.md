# 複製簡單請求

{{legacy-arb}}

複製簡單請求而非參考請求。 簡單請求是指未引用其他請求或儲存格內容的請求。

[參考請求](/help/analyze/legacy-report-builder/manage-requests/c-copy-requests/t-copy-referential-requests.md)會將儲存格的值視為參數的輸入值，如資料篩選或關聯篩選。 這些篩選器會使用比對或趨勢，而且會以先前請求的結果或使用者輸入的儲存格（稱為輸入儲存格）內容為基礎。

複製簡單請求

1. 建立有效的請求。
1. 以滑鼠右鍵按一下任一個已映射請求的儲存格，或選擇含請求的儲存格區域。

   請一致地從請求所涵蓋的儲存格群組中選取要複製的儲存格。 首選是請求所涵蓋之儲存格集的頂端和最左側的儲存格，並從左到右運作。 這是因為Excel試算表有數百欄和數千列可供向右和向下展開。 如果您決定從與請求相關的一組儲存格中最右邊或最下方的儲存格開始複製請求，則系統將不允許您貼上請求（如果要貼上的儲存格將超出試算表的左邊或上邊框）。
1. 選擇&#x200B;**[!UICONTROL 「複製請求」]**。
1. 在試算表的其他部分中，以滑鼠右鍵按一下空白的儲存格 (不含請求的儲存格)。

   為避免遺失或損毀您已建立的請求，您無法將包含請求的儲存格貼上至目前與請求對應的儲存格。 如果您複製或剪下含有請求的儲存格，則以滑鼠右鍵按一下含有請求的儲存格（或儲存格組）時，捷徑功能表無法顯示[!UICONTROL 貼上請求]選項。 您必須選取其他儲存格作為貼上操作的目的地，請求才能重疊。 無論您是選取含有貼上請求的單一儲存格，還是包含請求的儲存格區域，均適用。
1. 按一下&#x200B;**[!UICONTROL 「貼上請求」]**。

   系統會將原始請求的副本放置在位置與原始請求相對的儲存格內。

   >[!NOTE]
   >
   >系統只會複製請求，不會複製儲存格的內容。 如果您有其他跟請求無關，但與了解儲存格中顯示資料相關的資訊 (如表格欄標題或列識別碼)，請使用 Excel 的標準「複製」和「貼上」命令。

   由於Excel使用不同的剪貼簿來複製儲存格內容與複製請求，因此您可以先複製非請求儲存格內容，然後再執行序列中的複製/貼上及複製請求/貼上請求。 不過，如果您將格式套用至試算表中的請求，然後複製並貼上，Report Builder會重新產生原始格式（例如框線、字型等） 貼上區域中。

   在貼上請求之前修改您已經複製或剪下到剪貼簿的請求，會從剪貼簿中移除請求。 因此，若要讓請求保持其原始狀態，請勿在您複製請求與貼上請求之間修改請求。
