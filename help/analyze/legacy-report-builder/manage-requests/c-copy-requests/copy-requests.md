# 複製請求概觀

{{legacy-arb}}

您可以複製與多個請求相互映射的儲存格，然後再將內容貼到試算表內空白的選定區域中。

複製儲存格後，Report Builder和Excel會決定貼上最小複製數所需的區域。 如果區域夠大，貼上動作會建立所有請求的復本，其中每個貼上的請求都有與原始請求相同的空間排列和格式。

這稱為傳播要求。 這是建立長報表的簡單快速方法。 Report Builder會先在target貼上區域的儲存格中貼上所有請求，以傳播請求，然後根據為請求建立的報表日期重新整理儲存格。
