# 遞增排序或遞減排序

{{legacy-arb}}

您可以依照遞增或遞減順序來排序「請求管理員」中列出的請求。

按一下表格標頭中的欄名稱。 欄會藉由在表格標題中顯示三角形圖示來顯示排序順序。 如果三角形指向下方，則排序順序是遞減的（對於字母數字資料，是9-1和Z-A），而如果指向上方，則排序順序是遞增的（對於字母數字資料，是1-9和A-Z）。

預設的排序順序為遞增順序。
