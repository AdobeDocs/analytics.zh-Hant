# 系統需求

{{legacy-arb}}

系統建議與必要條件。

* Windows XP或更新版本
* 32位元或64位元Microsoft Excel 2007或更新版本
* 不支援OS X

**必備條件**

* 您組織的合約包括Report Builder for Excel。
* 您的使用者名稱屬於預先定義的Report Builder使用者群組。 這項需求包括 [!DNL Analytics] 管理員。
* 使用者必須執行 Windows 並以管理員身分登入。
