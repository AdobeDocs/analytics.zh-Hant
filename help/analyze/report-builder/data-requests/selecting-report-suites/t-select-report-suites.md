# 選擇報表套裝

協助您在 Excel 中選擇報表套裝的步驟。

1. In Excel, click **[!UICONTROL Add-Ins]**.
1. In the toolbar, click **[!UICONTROL Create]**.
1. 在視窗 [!UICONTROL Request Wizard: Step 1] 中，選取下方的報表套裝 [!UICONTROL Report Suite]。
