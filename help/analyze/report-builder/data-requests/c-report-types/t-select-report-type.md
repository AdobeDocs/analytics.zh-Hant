# 選取報表類型

為資料請求選擇基礎報表類型的步驟。

1. 導覽至[!UICONTROL 「請求精靈: 步驟 1」]視窗。
1.  在「**報表套裝軟體**」功能表中選擇報表套裝軟體。
1. 為資料請求選擇基礎報表類型。
