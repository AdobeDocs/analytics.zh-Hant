# 選擇報表類型

為資料請求選擇基礎報表類型的步驟。

1. 定位至 [!UICONTROL Request Wizard: Step 1] 窗口。
1. Select a report suite from the **[!UICONTROL Report Suite]** menu.
1. 為資料請求選擇基礎報表類型。
