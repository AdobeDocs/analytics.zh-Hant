# 遞延日期

若要自訂請求的遞延日期：

1. 在[!UICONTROL 「請求精靈: 步驟 1」]中選擇&#x200B;**[!UICONTROL 「遞延日期」]**。
1. 按一下&#x200B;**[!UICONTROL 「遞延」]**&#x200B;連結以選擇遞延類型，然後使用日期連結來設定遞延日期。

   [!UICONTROL 「遞延日期」]的設定會隨著您的選擇而改變。例如，當您選擇「[!UICONTROL 循週遞延]」時，Report Builder 會提供週專用的設定讓您選為遞延模式。

1. 視需要設定進階選項。
