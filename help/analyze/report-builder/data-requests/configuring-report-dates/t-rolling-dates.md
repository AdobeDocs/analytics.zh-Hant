# 滾動日期

說明為請求自訂遞延日期的步驟。

1. On the [!UICONTROL Request Wizard: Step 1], select **[!UICONTROL Rolling Dates]**.
1. 按一下「**遞延**」連結以選擇遞延類型，然後使用日期連結來設定遞延日期。

   「[!UICONTROL 遞延日期]」的設定會隨著您的選擇而改變。例如，當您選擇「[!UICONTROL 循週遞延]」時，Report Builder 會提供週專用的設定讓您選為遞延模式。

1. 視需要設定進階選項。
