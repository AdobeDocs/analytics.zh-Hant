# 將流量與商務度量合併為同一請求

您可以將流量與商務量度型別結合在一個請求中，而非建立個別請求。

流量和商務量度會顯示在「請求精靈：步驟2」的相同量度類別中。 群組類別會加上標籤 **[!UICONTROL 標準]**.
