# 結合相同請求中的流量和商務度量

現在，您可以將這兩種度量類型合併為同一個請求，而不須分別建立請求。

現在，在「請求精靈」的步驟 2 中，流量與商務度量會顯示在同一個度量類別。過去，這兩個度量分屬「商務」類別與「流量」類別。

The new grouped category is labeled **[!UICONTROL Standard]**. .
