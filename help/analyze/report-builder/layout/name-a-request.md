# 命名請求

您可以為請求指定自訂名稱，該名稱會顯示在「請求管理員」中。

For more information, see [Format Display Headers](../../../analyze/report-builder/layout/t-format-display-headers.md#task_45C7C4938C2C47FCB02634A1248AA831).
