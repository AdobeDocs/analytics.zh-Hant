# 設定自訂配置

「自訂配置」提供「樞紐配置」的大部分功能，而且能讓您選擇將格線中的各個項目放置在試算表中的位置。此項配置提供舊版中具備的彈性，包括新增劃分和 Microchart。

* [建立 Microchart](/help/analyze/report-builder/layout/t-create-a-microchart.md)
