# 報告建立工具概觀

 Report Builder 是 Microsoft Excel 的增益集。它可讓您從可插入Excel工作表的報告與分析資料建立自訂請求。 這些請求可動態參考工作表中的儲存格，而您可以更新及自訂 Report Builder 展示資料的方式。
