# Adobe Analytics 安全性資源

Adobe遵循已建立的安全性程式與控制機制，以符合業界接受的標準、法規與認證。 另請參閱 [Adobe安全性](https://www.adobe.com/trust/security.html) 以取得Adobe公司安全性實務的詳細資訊。

有關Adobe Analytics專屬實務的資訊，請參閱 [Adobe Analytics安全性總覽](https://www.adobe.com/content/dam/cc/en/trust-center/ungated/whitepapers/experience-cloud/adb-analytics-security-wp.pdf). 它概述了圍繞架構、驗證、資料中心、網路和人員所遵循的流程。

另請參閱 [Adobe的Web服務弱點揭露計畫](https://hackerone.com/adobe) 如果您在HackerOne上發現漏洞，需要回報任何Adobe產品，包括Adobe Analytics。
