# Adobe Analytics使用的IP和網域

有些防火牆設定會封鎖源自Adobe資料收集伺服器或負責存取資料的伺服器的IP位址。 您可以使用此範圍清單來變更組織的防火牆設定，以允許存取並從組織內傳送資料。

>[!IMPORTANT]
>
>雖然Adobe盡其所能讓本檔案保持最新狀態，但無法保證IP範圍清單保持不變。 可能的變更包括業務的成長與擴充、網際網路註冊需要變更Adobe的IP位址空間，或網際網路服務供應商停止運作。

## 允許相關技術域

Adobe Analytics使用下列主機來改善效能和產品體驗。 Adobe建議將這些網域新增至防火牆的允許清單，以獲得使用Adobe Analytics的最佳體驗。

| 技術 | 網域 |
| --- | --- |
| Adobe Analytics網域 | `adobe.com`, `adobe.net`, `adobe.io` |
| Adobe Analytics舊網域 | `omniture.com` |
| Amazon AWS | `aaui-879784980514.s3.us-east-2.amazonaws.com` |
| Amazon CloudFront | `d30ln29764hddd.cloudfront.net` |
| Gainsight | `esp.aptrinsic.com` |
| 啟動Darkly | `app.launchdarkly.com` |
