# Adobe Analytics使用的IP和網域

有些防火牆設定會封鎖來自 Adobe 的資料收集伺服器或負責存取資料的伺服器 IP 位址。您可以使用此範圍清單來變更組織的防火牆設定，以允許存取並在組織內傳送資料。

>[!IMPORTANT]
>
>Adobe 會盡可能即時更新這份文件，但無法保證 IP 範圍清單保持相同。可能的變更包括企業成長與擴充、網際網路註冊管理機構須變更 Adobe 的 IP 位址空間，或網際網路服務供應商停止營運。
