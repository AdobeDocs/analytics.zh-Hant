# 工作流程概觀

如何佈建、共用以及使用對象資料。

| 步驟 # | 工作說明 | 附註 |
|--- |--- |--- |
| 步驟 1 | 開啟伺服器端轉送。 | 請參閱 管理 [員使用指南中的](/help/admin/admin/c-server-side-forwarding/ssf.md) 「伺服器端轉送」。 |
| 步驟 2 | 在 AAM 中設定 Analytics 目的地。 | 請參閱[建立 Analytics 目的地](https://docs.adobe.com/content/help/en/audience-manager/user-guide/features/destinations/experience-cloud-destinations/create-analytics-destination.html)。 |
| 步驟 3 | [在 Analytics 中使用資料](/help/integrate/c-audience-analytics/c-workflow/use-audience-data-analytics.md)。 | 您可以在 Analysis Workspace 中使用新的 AAM「對象」維度及其相關功能 (「區段比較」、「流量」、「文氏圖表」等)。<br>「對象」維度也適用於 Reports &amp; Analytics、Ad Hoc Analysis、Report Builder、Data Warehouse 及「資料摘要」。<br>**注意&#x200B;**：「資料摘要」僅包含「對象 ID」，不含「對象名稱」。 |
