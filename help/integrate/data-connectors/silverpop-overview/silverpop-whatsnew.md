# Updates for Silverpop 2.0{#updates-for-silverpop}

下列功能是Silverpop Data Connectors整合2.0版的新功能：

* 量度匯入頻率從每日增加至每小時。
* 非零售客戶的兩個額外標準度量。
* 重新行銷給可能未透過電子郵件按一下的使用者。
* 產品分類會匯出至Silverpop，以便在再行銷訊息中進行其他自訂。
* 再行銷區段中提供額外資料(訂購、收入和件數)。
