# Integration Prerequisites{#integration-prerequisites}

列出Adobe Analytics客戶的必要條件。

## Prerequisites for Adobe Customers {#section-23ba9dd1a1774f3099a6764d3fadd4ef}

* 必須是Adobe Analytics的目前客戶。
* 必須擁有管理員權限。
* 報表套裝中必須有個可用且啓用的eVar。
* 報表套裝中必須啓用個可用事件。
