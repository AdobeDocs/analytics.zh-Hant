# Key Benefits and Features{#key-benefits-and-features}

此項整合提供下列主要優點。

* 每日自動同步AppFigures和Adobe Analytics之間的應用程式商店維度和度量。
* 整合啓動前最多13個月的歷史資料回填。

   >[!NOTE]
   >
   >如果您先前曾與App Store資料供應商整合以避免重疊，請與AppFigures連絡。

* 應用程式商店資料的專屬變數，因此您不需要使用自訂商務變數和事件。
