# Adobe 整合: 夜間資料匯入{#adobe-integration-nightly-data-import}

整合的資料收集部分，會收集關於網站訪客的點進和閱覽資料。為了取得 DFA 點按、曝光數和成本量度，Google 與 Adobe 合作開發出夜間程序，可將這項額外的資料匯入至整合的報表套裝中。這些量度會透過資料來源匯入，這表示，這些量度只會以彙整的形式提供，而不會以造訪層級提供。請參閱[整合功能](../dfa-data-connector-analytics/dfa-integration-features.md#concept-ff93289d1662410e98f62c200394b3e3)以取得詳細資訊。
