# 報表套裝範本概觀

報表套裝範本會設定數種報表套裝最常用的設定。

若要在建立新報表套裝時節省時間，您可以選取與您設計的報表套裝組態類似的範本。 選取報表套裝範本並不會限制您的實作方式。 報表套裝範本所做的任何設定，都可以在報表套裝建立後變更。
