# 預設量度

[!UICONTROL **編輯設定**] > [!UICONTROL **個別報告套裝設定**] > [!UICONTROL **預設量度**]

除非使用者選取自訂量度集，否則Analytics在所有報表和專案中會顯示預設（現成）量度集。 選取的量度會顯示給相關報表套裝的所有使用者。 您一次只能更新一個報表套裝的預設量度。 這些設定無法保證預設量度的顯示順序。

有關詳細資訊，請參閱：

* [Analysis Workspace 中的量度](/help/analyze/analysis-workspace/components/apply-create-metrics.md)

* [計算量度](/help/components/calculated-metrics/cm-overview.md)
