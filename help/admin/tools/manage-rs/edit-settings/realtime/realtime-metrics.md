# 支援的即時量度和維度

列出即時報表支援的量度和維度。 如果已定義分類，也支援分類。

## 支援的量度 {#metrics}

* 收入
* 訂購
* 單位數
* 購物車
* 購物車檢視
* 結帳
* 購物車新增
* 購物車移除
* 實例
* 自訂事件 1-100 (若已在報表套裝中啟用)

## 支援的維度 {#dimensions}

* 頁面
* 搜尋關鍵字
* 搜尋引擎
* 反向連結網域
* 地域劃分國家/地區
* 地域劃分割槽域
* 地域劃分城市
* 地域劃分U.S. DMA
* 產品
* 伺服器
* 網站區段
* 追蹤程式碼
* 自訂連結
* 檔案下載
* 退出連結
* Custom Insight 變數 (prop) 1-75 (若已在報表套裝中啟用)
