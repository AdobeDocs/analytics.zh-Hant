# 自訂報表說明

您可以提供eVar、prop、分類和事件的自訂報表說明。 所有一般使用者都能看到說明。

**[!UICONTROL 「Analytics]** > **[!UICONTROL 管理員]** > **[!UICONTROL 報表套裝]** > **[!UICONTROL 編輯設定]** > **[!UICONTROL *變數」*]**

>[!NOTE]
>
>新增說明適用於大部分傳統分類，但目前不適用於行動裝置分類。

![](/help/admin/tools/assets/report_descriptions.png)

1. 提供報表的說明。 字元數上限為255。
1. 按一下&#x200B;**[!UICONTROL 「儲存」]**。

說明會顯示在兩個位置：

* 報告的標題列：

  ![](/help/admin/tools/assets/report_description_2.png)

* 當您將滑鼠游標停留在左側導覽中的報表上時，提供工具提示：

  ![](/help/admin/tools/assets/report_description_3.png)
