# 應用程式分類

在您啟用[應用程式報告](app-reporting.md)下的任何功能後，行動裝置分類便會啟用。 它提供與[轉換分類](conversion-var-admin/conversion-classifications.md)或[流量分類](c-traffic-variables/traffic-classifications.md)相同的介面，但針對應用程式報表維度。
