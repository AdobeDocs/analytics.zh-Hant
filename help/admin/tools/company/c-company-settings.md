# 公司設定概觀

[!UICONTROL 公司設定]頁面可讓您設定要對您的組織所管理的所有報表套裝套用的設定。

**[!UICONTROL Analytics]** > **[!UICONTROL 管理員]** > **[!UICONTROL 所有管理員]** > **[!UICONTROL 公司設定]**

+ [安全管理員](security-manager.md)
+ 支援資訊：當您需要聯絡管理員時，「支援資訊」頁面會管理在整個Adobe Analytics中顯示的支援資訊。 例如，您可能無法存取您嘗試存取的功能。
+ [Report Builder 報表](report-builder-reports-admin.md)
+ [單一登入](single-signon-admin.md)
+ [待定動作](pending-actions-admin.md)
+ 品牌結合：管理品牌結合影像頁面可讓您在報表和UI中顯示自己公司的標誌。 按一下&#x200B;**[!UICONTROL 啟用品牌結合]**。
+ [隱藏報告套裝](c-hide-report-suites.md)
+ [檢視功能存取層級](feature-access-levels.md)
+ [網路服務](web-services-admin.md)
+ 中國效能最佳化：Adobe在中國大陸的效能最佳化元件可讓中國的客戶直接傳送資料給中國邊緣節點，而不是世界上的其他地方。 比起將資料傳送至中國境外的節點，這麼做可縮短頁面載入時間，並提升資料準確度。 如需詳細資訊，請參閱UI中的常見問題區段。
