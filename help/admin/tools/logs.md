# 記錄檔

記錄檔可協助您查看使用者何時登入、其使用情形、存取權、報告套裝以及管理員變更。

**[!UICONTROL Analytics]** > **[!UICONTROL 管理員]** > **[!UICONTROL 所有管理員]** > **[!UICONTROL 記錄檔]**

## 管理記錄檔 {#section_8ADE8A7204A8401C968ABC20AECA381D}

管理記錄檔會報告管理員在管理工具中所做的一切變更。 此記錄提供從三個記錄中的任何一個存取用戶定義的報表的管道。 您可以在指定的日期範圍內搜尋符合您選取條件的事件。

## 使用與存取記錄檔 {#section_6FBAF92D9EA244809C45A78A2F0A7232}

[!UICONTROL 使用情況和存取記錄檔]可讓您評估使用者帳戶層級的報告使用情形。 例如，可追蹤Analysis Workspace中的開啟、建立、更新、取消共用和刪除動作。 讓您更清楚掌握正在使用 Workspace 的使用者以及 Workspace 的使用頻率。

| 元素 | 說明 |
|---|---|
| 日期範圍 | 指定日期範圍篩選。 您可以手動輸入YYYY-MM-DD格式的日期，或按一下「行事曆」圖示以選取日期。 |
| 登入 | 依使用者名稱篩選記錄。 |
| IP | 依IP位址篩選記錄。 |
| 報表套裝 | 依特定報表套裝ID篩選記錄。 |
| 事件類型 | 依事件型別篩選記錄。 從下拉式清單中選取事件型別。 請參閱以下的完整事件型別清單。 |
| 事件 | 依事件說明中的字詞或片語篩選記錄。 |
| 下載報表 | 將[!UICONTROL 使用與存取記錄檔]的內容匯出至以Tab分隔的檔案。 |

## 報表套裝變更記錄 {#section_3864966639414BBEA871F4D0352F56B6}

報表套裝變更記錄檔會顯示在管理員以外對報表套裝所做的變更。

可從「[!UICONTROL 管理工具]」外部修改報表套裝的工具包括：

* 在網頁瀏覽器中進行的分類上傳（透過FTP進行的分類上傳不包含在變更記錄中）
* 在舊版中進行的變更。
* 客戶代表或客戶服務使用內部工具進行的變更

| 元素 | 說明 |
|---|---|
| 日期範圍 | 指定日期範圍篩選。 您可以手動輸入YYYY-MM-DD格式的日期，或按一下「行事曆」圖示以選取日期。 |
| 公司 | 依公司名稱篩選記錄。 |
| 登入 | 依使用者名稱篩選記錄。 |
| IP | 依IP位址篩選記錄。 |
| 事件 | 依事件說明中的字詞或片語篩選記錄。 |
| 下載報表 | 將[!UICONTROL 使用與存取記錄檔]的內容匯出至以Tab分隔的檔案。 |
