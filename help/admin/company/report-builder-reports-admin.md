# Report Builder 報表

管理指派給Report Builder使用者的授權。

指派給使用者的每個Report Builder授權都會自動分配十個排程任務。 您可以調整並重新配置工作的數目。排程的工作不能超過每個使用者可用的工作數目。過度排程的工作將無法執行。

您可以將Report Builder存取權授與 [!DNL Analytics] 使用者：在 [Adobe Admin Console](/help/admin/admin-console/home.md).
