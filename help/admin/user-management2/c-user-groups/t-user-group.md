# 新增使用者群組

>[!IMPORTANT]
>
>User and product management is moving to the [Admin Console](https://helpx.adobe.com/enterprise/using/admin-console.html). 輪到您移轉使用者時，Adobe 會通知您。After all customers have migrated, help content for **[!UICONTROL Analytics]** &gt; **[!UICONTROL Admin Tools]** &gt; **[!UICONTROL User Management]** will be retired.

說明如何新增使用者群組的步驟。

1. Click **[!UICONTROL Analytics]** &gt; **[!UICONTROL Admin]** &gt; **[!UICONTROL User Management]**.
1. Click **[!UICONTROL Add New User Group]**, then complete the fields and options described in [Define User Groups](../../../admin/user-management2/c-user-groups/groups.md#concept_DBBCB8275D554B02B4B4E1168BA69A05).

   快速連結:

   * [分析工具](../../../admin/user-management2/c-customize-report-access/groups-analytics-tools.md#concept_C4383A6C0F5E4130875FDD3756F2E2FC): 啟用「一般」項目 (帳單、日誌等)、公司管理、工具、網站服務存取、Report Builder 和 Data Connectors 整合的使用者權限。
   * [報表套裝工具](../../../admin/user-management2/c-customize-report-access/groups-report-suite-tools.md#concept_C94E9864349B428AB9CCE0CA4B0A40FF): 為 API 存取、報表套裝管理、工具和報告與控制面板項目啟用使用者權限。
   * [量度](../../../admin/user-management2/c-customize-report-access/groups-metrics.md#concept_05D54436430E4320A48C7C685D337FBE): 為流量、轉換、自訂事件、解決方案事件、內容感知等等啟用權限。
   * [維度](../../../admin/user-management2/c-customize-report-access/groups-dimensions.md#concept_68B36161345341369B6D01DC7DD42A22): 在粒度層級自訂使用者存取，包括 eVar、流量報表、解決方案報告和路徑報告。

1. Click **[!UICONTROL Save Group]** to create the group and return to the [!UICONTROL Groups] page.
