# 使用者和產品管理

在 Admin Console 中管理 Analytics 使用者、群組和產品。

>[!IMPORTANT]
>
>使用者和產品管理已移至 [Adobe Admin Console](https://helpx.adobe.com/tw/enterprise/using/admin-console.html). 輪到您移轉使用者時，Adobe 會通知您。
