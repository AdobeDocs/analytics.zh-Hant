# 報表套裝範本概述

報表套裝範本為數種報表套裝設定最常用的設定。

若要節省建立新報表套裝的時間，可以選取類似於您已設計之報表套裝設定的範本。選取報表套裝範本並不會限制您的實施方式。報表套裝範本中的任何設定，在建立報表套裝後都可加以變更。
