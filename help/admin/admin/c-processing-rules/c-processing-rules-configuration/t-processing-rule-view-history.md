# 檢視處理規則歷史記錄

可在「歷史」中檢視處理規則的各項變動。

1. 按一下&#x200B;**[!UICONTROL 「Analytics]** > **[!UICONTROL 管理]** > **[!UICONTROL 報表套裝」]**。
1. 選取報表套裝。
1.  按一下&#x200B;**[!UICONTROL 「編輯設定]** > **[!UICONTROL 一般]** > **[!UICONTROL 處理規則」]**。

1. 在&#x200B;**[!UICONTROL 處理規則]**&#x200B;頁面上，按一下&#x200B;**[!UICONTROL 「檢視歷史記錄」]**&#x200B;標籤。
