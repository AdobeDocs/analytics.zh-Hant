# 流量變數 (prop) 概觀

自訂分析流量變數 (或 prop) 可讓您為自訂資料與特定流量相關的事件建立相互關聯。prop 變數會嵌入至網站每個頁面上的實施代碼中。

使用「[!UICONTROL 流量變數]」頁面來啟用、停用或重新命名流量變數報告。一旦停用之後，流量變數的報告就不會在使用者介面裡顯示。

請參閱 也 [是實作](../../../implement/vars/page-vars/prop.md) 使用指南中的prop。
