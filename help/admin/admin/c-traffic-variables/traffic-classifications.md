# 流量分類

流量分類可讓您對流量變數 (屬性) 進行分類。流量分類只能使用文字分類。

「流量分類」頁面可讓您建立選定報表套裝的分類。分類後，透過關鍵資料產生的任何報告，亦可透過相關聯的屬性產生。

啟用分類後，請使用[分類匯入工具](/help/components/classifications/importer/c-working-with-saint.md)，將特定值指派給適當的分類。
