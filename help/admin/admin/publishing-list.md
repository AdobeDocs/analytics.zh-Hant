# 發佈清單

「發佈清單」可讓您輕鬆地傳送您的組織內不同群組所專用的各種報告，而無須個別建立數個排程報告。若您具有區位性的報表套裝，而想為每個部門分別提供一份特定的控制面板，「發佈清單」將可派上用場。或者，若您處理的是單一報表套裝，您可以使用發佈清單將資料傳送給多人，而無須個別鍵入其電子郵件地址。

在排程報告時，可指定多重發佈清單。
