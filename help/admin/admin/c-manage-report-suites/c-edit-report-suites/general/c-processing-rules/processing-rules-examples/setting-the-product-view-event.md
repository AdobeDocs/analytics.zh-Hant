# 從產品概觀頁面設定產品檢視事件

可以根據頁面值或根據變數中的特定值來設定事件。

| 規則集 | 值 |
|---|---|
| 條件 | 如果頁面名稱等於產品概述 |
| 動作 | 將事件產品檢視事件設定為自訂值 1 |
