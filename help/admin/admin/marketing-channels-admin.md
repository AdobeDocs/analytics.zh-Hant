# 行銷管道

「行銷管道」常用來提供訪客如何進入您的網站的相關資訊。您可以根據所要追蹤的管道以及追蹤方式，建立及自訂行銷管道處理規則。

**[!UICONTROL Analytics]** &gt;管 **[!UICONTROL 理]** &gt;報表套裝 **[!UICONTROL &gt;設定]** &gt;編輯行銷管 ********&#x200B;道。

請參閱[行銷管道](/help/components/c-marketing-channels/c-overview.md)。
