# 群組管理

群組是一組您想要給予一般存取權和權限集的使用者。例如，若您有 50 位行銷報告使用者，其中只有 20 位需要存取 Ad Hoc Analysis，則您可為這 20 位使用者建立群組。

如需相關設定資訊，請參閱[群組](/help/admin/user-management2/c-user-groups/groups.md)。
