# Report Builder 傳遞的報表

使用適用於 Microsoft Excel 的 Adobe Report Builder 增益集，可從網站分析資料 (您可將其插入至 Excel 工作表) 建立自訂請求。

請求可動態參考工作表中的儲存格，而且您可以更新及自訂資料的呈現方式。 針對FTP或SFTP，您必須確保主機可在外部使用。 如需[!UICONTROL Report Builder]傳遞之報告的詳細資訊，請參閱[Report Builder說明](/help/analyze/report-builder/rb-overview.md)。

如需FTP或SFTP限制和資料保留的資訊，請參閱[FTP和SFTP限制和資料保留](/help/export/ftp-and-sftp/ftp-limits.md)。
