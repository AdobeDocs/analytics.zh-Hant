# Data Warehouse 傳遞的報表

>[!NOTE]
>
>以下資訊與FTP和SFTP目的地型別有關。 FTP和SFTP是舊有的目的地型別。 設定Data Warehouse報表時，您應使用較安全的雲端目的地型別。 如需為Data Warehouse報表設定雲端目的地型別的詳細資訊，請參閱[建立Data Warehouse請求](/help/export/data-warehouse/create-request/t-dw-create-request.md)。

Data Warehouse 讓您可以根據您的獨特問題，請求顯示來自原始資料的進階資料關係的報表。

在這種情況下，FTP傳送可作為[!UICONTROL 進階傳送選項]設定的一部分使用。 若使用FTP，您需要現有的內部FTP帳戶或Adobe FTP帳戶。 Adobe不會透過Data Warehouse介面自動為您設定FTP帳戶。 FTP傳送應立即進行，視檔案大小和正在處理的報表數目而定。

如需FTP限制和資料保留的詳細資訊，請參閱[FTP限制和資料保留](/help/export/ftp-and-sftp/ftp-limits.md)。
