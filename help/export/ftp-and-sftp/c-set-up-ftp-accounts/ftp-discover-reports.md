# Ad Hoc Analysis 傳遞的報表

您可設定傳遞排程，以自動將報表傳送至電子郵件地址或 FTP 伺服器。

傳遞排程可在一段選定期間內進行。報表通常在隨後期間的第一天傳送，最遲不超過第二天早上。您隨時可修改或停止這些已排程的傳遞。如需[!UICONTROL 臨機分析]傳遞的報表的詳細資訊，請參閱[臨機分析說明](https://experienceleague.adobe.com/docs/analytics/analyze/ad-hoc-analysis/adhoc-home.html?lang=zh-Hant#Discover_Help)。

如需 FTP 限制和資料保留的相關資訊，請參閱 [FTP 限制和資料保留](/help/export/ftp-and-sftp/ftp-limits.md)。
