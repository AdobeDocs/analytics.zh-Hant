# 使用 SFTP 傳送 Adobe 資料至外部 FTP 帳戶

Adobe 工程技術服務小組可以設定自訂專案，使用 SFTP 將您的資料重新導向至外部 FTP 帳戶。

該值就是安全地傳送給您的資料，無須從 Adobe FTP 站台擷取資料。若想確定專案範圍，請聯絡 Adobe 客戶經理。
