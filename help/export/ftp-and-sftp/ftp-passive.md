# 使用被動式 FTP 模式

主動式 FTP 和被動式 FTP 會控制連接埠連線的建立方式，且選項與防火牆有一些關聯。

Adobe 使用被動式 FTP，因為這是比較安全的資料傳輸模式，資料流程是由檔案傳輸程式 (FTP) 用戶端設定與起始，而非 FTP 伺服器程式。如果您在連線至AdobeFTP時遇到問題，請確定您使用被動模式。
