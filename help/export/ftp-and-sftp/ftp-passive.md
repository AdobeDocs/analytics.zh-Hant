# 使用被動式 FTP 模式

主動式 FTP 和被動式 FTP 會控制連接埠連線的建立方式，且選項與防火牆有一些關聯。

Adobe使用被動式FTP，這是更安全的資料傳輸形式，資料流程由檔案傳輸程式(FTP)使用者端設定，而非FTP伺服器程式啟動。 如果您連線至Adobe FTP時遇到困難，請務必使用被動模式。
