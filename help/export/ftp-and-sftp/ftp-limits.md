# FTP和SFTP限制及資料保留

所有Adobe FTP和SFTP帳戶都具備100 GB的資料儲存限制。 檔案會在這些 FTP 帳戶中保留 90 天。

## 同時連線的限制

* 每個來源 IP 與伺服器的同時連線不能超過 50 個
* 每個來源 IP 在 60 秒以內與伺服器的新連線不能超過 20 個
