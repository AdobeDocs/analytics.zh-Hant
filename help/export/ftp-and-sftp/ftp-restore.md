# 還原已刪除的FTP或SFTP帳戶和資料

Adobe無法還原已從FTP或SFTP帳戶刪除的任何資料。

如果您打算使用的FTP或SFTP帳戶已停用，請聯絡您的Adobe帳戶團隊，他們可與您討論其他選項。
