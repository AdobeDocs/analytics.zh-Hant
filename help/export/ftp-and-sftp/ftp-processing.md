# FTP和SFTP處理時間

任何 FTP 檔案 (包括分類或資料來源) 的處理時間取決於處理佇列中的檔案數以及檔案大小。

如果您提交的檔案大於5MB，匯入時間可能會稍長一些。 如果您在1-2天後沒有在FTP站台中看到已完成的檔案，但報表工具並未反映已上傳的資料，則貴組織的受支援使用者應聯絡Adobe客戶服務，判斷檔案是否有問題。

建立FTP檔案時，請務必新增通知收件者的電子郵件地址。 如此一來，大多數問題都會導致通知傳送給收件者。
