# Adobe Analytics 技術文件更新

* 如需新功能、管理員注意事項和錯誤修正的完整清單，請參閱最新的 [Experience cloud發行說明](https://marketing.adobe.com/resources/help/en_US/whatsnew/)。
* 如需有關 Analysis Workspace 新功能的詳細資訊，請參閱[此主題](/help/analyze/analysis-workspace/new-features-in-analysis-workspace.md)。
* 如需有關Adobe Analytics for Audio and Video的詳細資訊，請參閱「在Analytics [中測量音訊和視訊」](https://docs.adobe.com/content/help/en/media-analytics/using/media-overview.html)。
