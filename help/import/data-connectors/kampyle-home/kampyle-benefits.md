# 主要優點和功能{#key-benefits-and-features}

* 每日在 Kampyle 和 Adobe Analytics 之間自動同步意見維度和量度。
* 新的流量和轉換報表，內容包括意見計數、評分和其他評論。
* 新的意見維度，可用來建立區段以切入任何 Adobe Analytics 報表。
