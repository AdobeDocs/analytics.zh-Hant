# 主要優點和功能{#key-benefits-and-features}

* Kampyle與Adobe Analytics之間每日自動同步意見維度與量度。
* 新的流量和轉換報告，包括意見計數、評分和其他意見。
* 可用來建立細分的新意見回應維度，以便深入任何Adobe Analytics報表。
