# Analytics 匯入指南

![橫幅](../../assets/doc_banner_import.png)

本指南說明如何將資料取得Analytics。其功能包括:

* **資料來源：** 將檔案上傳至指定的Adobe FTP站台。Adobe會擷取檔案並包含報表套裝中的資料。
* **資料插入API：** 透過API直接將資料傳送至Adobe的資料收集。
* **資料連接器：** 與支援的第三方合作，將其資料匯入Adobe Analytics，豐富您的報表套裝。
