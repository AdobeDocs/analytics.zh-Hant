# 資料來源處理時間

>[!Note]
>任何資料處理時段都應視為約略時間，不構成服務等級協議 (SLA)。

資料處理時間

* 當天的資料：資料上傳後約 2 小時處理完成。
* 前一天的資料: 資料上傳後約 3 小時處理完成。

上傳中每增加一天，處理時間約需增加 1 小時，最多達 17 小時。

例如，如果您上傳前一天的資料，約 2 小時後將可在 Analytics 中看到這筆資料。而如果上傳上一個月的資料，約需 17 小時後可在 Analytics 中看到這筆資料。
