# 瀏覽器類型

指的是訪客所使用的瀏覽器類型，例如 Internet Explorer、Mozilla Firefox、Google Chrome 等。

無法辨識的瀏覽器類型會顯示為「未指定」。

例如：

| Google |
|---|
| Mozilla |
| Microsoft |
| Apple |
| Opera |
| 未指定 |
| BlackBerry |
| UC Web |
| 其他 |
| Nokia |
