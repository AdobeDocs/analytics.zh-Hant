# 指定市場區域 (DMA)

將美國區隔至行銷區域。每個市場區域的「Internet 服務提供者 (ISP)」為美國 Internet 名稱註冊管理機構 (ARIN) 提供其使用的 IP 位址。Adobe 與 Digital Envoy 合作，取得將網站訪客所使用的 IP 位址與該位址的所在城市、州、郵遞區號和 DMA 進行比對的地域劃分資料。
