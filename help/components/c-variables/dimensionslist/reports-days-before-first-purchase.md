# 首次購買前的天數差距

顯示客戶首次造訪網站和最後消費之間的間隔天數。

例如，若某個訪客在首次瀏覽的隔天購買了商品，則在該訪客 Cookie 剩餘的有效期間內，任何瀏覽或事件都將顯示在報告中的 *1 天*&#x200B;明細項目上。
