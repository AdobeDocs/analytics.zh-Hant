# 獨特客戶

購買商品的獨特訪客。「獨特客戶」在該時間範圍內只會計入一次，而不會再次計入，無論該訪客回訪並購買了多少次。「獨特訪客」只會在指定時段內的第一次瀏覽時計入，且在該時段到期前將不會再次計入。在該時段到期後，即會再次對獨特訪客計數。「獨特客戶」一律計為「獨特訪客」，因為他們必須瀏覽網站才能購物。

另外也可使用以時段為基礎的獨特客戶 (每小時、每日、每週等) 報告，此報告類似於以時段為基礎的獨特訪客報告。
