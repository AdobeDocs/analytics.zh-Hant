# 網路通訊協定

依照行動裝置支援的網路通訊協定 (GPRS、CSD、EDGE、HSCSD 等)，將行動裝置分組。該報告還包括無法識別網路通訊協定的「未知」群組。

訪客在使用行動裝置存取網站時，「網路通訊協定」報告將至少增加一。如果電話支援多種網路通訊協定，那麼報告中的每個群組都會隨之增加。因此，群組的總數可能比顯示在報告底部的總數大。
