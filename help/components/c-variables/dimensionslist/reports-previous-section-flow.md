# 上一區域流量

「上一區域流量」報表類似於「上一頁流量」報表。但顯示的是網站區域 (相關網頁的群組) 的資料。若一個頁面包含於多個網站區域，那麼報告顯示所有網站區域的資料。

例如，線上零售商可能擁有分別針對其產品和產品品牌的網站區域。在這種情況下，一個產品網頁可能歸入多個網站區域。雖然某個產品頁面僅被檢視過一次，「上一區域流量」報告會顯示與該頁面有關的每個網站區域的頁面檢視情況。
