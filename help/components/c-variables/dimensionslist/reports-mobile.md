# 行動

顯示從行動裝置存取網站的相關資訊。 行動報表會收集用於檢視您網站的行動裝置的相關資料，以改善訪客識別。 行動報表可識別裝置的製造商和功能，例如螢幕大小、視訊、音訊和Cookie支援，以及其他度量。 「行動裝置」報告適用於所有報表套裝。

## 行動 {#topic_D306EA4558194488AC47A45B9C570150}

顯示從行動裝置存取網站的相關資訊。 行動報表會收集用於檢視您網站的行動裝置的相關資料，以改善訪客識別。 行動報表可識別裝置的製造商和功能，例如螢幕大小、視訊、音訊和Cookie支援，以及其他度量。 「行動裝置」報告適用於所有報表套裝。

>[!NOTE] 行動報表只會顯示行動網路流量。使用原生iOS、Android和其他SDK收集的行動應用程式資料會顯示在「行動應用程式報表」中，且只有在「管理工具」中啟用時才會顯示。
