# 影像支援

依照行動裝置支援的影像類型對行動裝置分組。例如，如果訪客的電話支援 .jpg 影像並存取了網站，則「影像支援」報告至少增加一。如果電話支援更多音訊格式，那麼瀏覽網站可以導致該次瀏覽增加多次。換言之，如果您的電話支援 .jpg、.png 及 .gif 格式，報告中這些群組的每一組都將增加。因此，群組的總數可能比顯示在報告底部的總數大。
