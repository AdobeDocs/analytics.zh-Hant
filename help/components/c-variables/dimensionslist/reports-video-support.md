# 視訊支援

依照行動裝置支援的影片格式類型對行動裝置分組。例如，如果訪客的行動裝置支援 .mp4 格式，則當其存取您的網站時，「視訊支援」報告至少增加一。如果電話支援多種影片格式 (如 .mp4 和 .wmv)，則報告中的每一群組都會隨之增加。因此，群組的總數可能比顯示在報告底部的總數大。
