# 購物車

顯示特定時段內開啟的購物車數量。您可以執行報告來分析購物車檢視、購物車新增、購物車移除和結帳。購物車通常在客戶選擇購買項目時開啟，但在沒有項目時亦可開啟。

您可以使用[!UICONTROL 購物車]:

* 確定網站上已開啟購物車的模式及其數量的最高值或最低值。
* 檢查特定時段，瞭解對開啟購物車貢獻突出的量度。
