# 維度說明

「Reports &amp; Analytics」中可用的預先定義報告的定義，以及其他 Analytics 工具 (例如 Analysis Workspace) 中可用的維度。

維度是量度資料的描述或特徵，可以在報表中加以檢視、劃分及比較。它們是經過劃分的非數值和日期，或原始報表類型量度的分類。
