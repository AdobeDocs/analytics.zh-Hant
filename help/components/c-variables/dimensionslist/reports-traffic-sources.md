# 流量來源

顯示促進網站流量的網站位置。您可以看到您網域外部的哪些搜尋引擎和網站為您帶來訪客。

## 流量來源 {#topic_6718F8EFD5984DC5839B9E8F6CC45EDA}

顯示促進網站流量的網站位置。您可以看到您網域外部的哪些搜尋引擎和網站為您帶來訪客。

此功能表中的報告分為三個基本類別:

* 搜尋引擎
* 搜尋關鍵字
* 反向連結和反向連結網域
