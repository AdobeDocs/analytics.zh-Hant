# 色彩深度

依據支援的色彩數目對行動裝置的點擊分組。該報告/維度顯示使用行動裝置瀏覽您網站的訪客總數，並依照他們行動裝置中配置的色彩數目對訪客進行分組。例如，如果訪客的行動電話支援 24 色，則報告會增加與 24 色相對應的行項目。
