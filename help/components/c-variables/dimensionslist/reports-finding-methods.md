# 尋找方法

顯示以何種方式接收轉換成功事件的評價。例如，如果某個訪客透過搜尋引擎進入您的網站並消費，則由「尋找方法」指定搜尋引擎接收反向連結評價的方式。

請參閱[尋找方法](/help/admin/admin/finding-methods.md)以取得設定資訊。
