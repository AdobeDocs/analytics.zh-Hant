# 點擊類型

「點擊類型」維度可表示收集點擊時，應用程式位於前景或背景。

SDK 會自動收集此維度。SDK 4.13.6 版及更高版本均支援此維度。

如果勾選「停用背景點按數的舊式報告和屬性」，則背景點擊數僅會顯示在[虛擬報表套裝](../../../components/vrs/vrs-mobile-visit-processing.md#concept_EC51308E4FD14E149F1B5D63C0AB34BD)。
