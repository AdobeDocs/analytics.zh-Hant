# 資訊服務

依照行動裝置支援的新聞服務 (Channel I/B、EZ News Flash、W+INFO 等) 對行動裝置實行分組。訪客在使用行動裝置存取網站時，「資訊服務」報告將至少增加一。如果電話支援多種新聞服務，那麼報告中的每個群組都會隨之增加。因此，群組的總數可能比顯示在報告底部的總數大。
