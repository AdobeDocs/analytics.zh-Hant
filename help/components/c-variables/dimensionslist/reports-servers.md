# 伺服器

可讓您將某部特定伺服器代管的頁面分組。列出您網站中的所有伺服器，同時也會告訴您哪些伺服器最常被存取。例如，如果您的網站由兩部不同的伺服器代管，您可能會想要瞭解其中一部伺服器就曝光數而言是否比另一部更高。
