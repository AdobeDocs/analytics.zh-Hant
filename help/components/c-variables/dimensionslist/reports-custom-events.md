# 自訂事件

您希望在訪客完成的網站上轉換動作。這些動作可能是註冊、訂閱、填寫銷售機會表單、啟動聊天、購買、預訂或完成調查。

由於每個報表套裝都略有不同，每個客戶使用本組報告的方式也不同。「自訂事件」報告可用作顯示事件發生次數的計數器。例如，如果設定 **[!UICONTROL event 1]** 為計算下載某個文件的次數，則 event 1 的自訂事件報告會顯示發生該事件 (或下載) 的總次數。您可以具有多個自訂事件報表。
