# JavaScript 版本

依據行動裝置支援的 Java 版本對行動裝置分組。「JavaScript 版本」報表會透過裝置所支援的「Java 規格請求」(JSR) 來識別 Java 版本。

您可以在 [Java 社區進程](https://jcp.org/en/jsr/overview)網站尋找更多有關指定 JSR 的資訊。
