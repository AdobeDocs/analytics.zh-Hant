# 訪問次數

協助您透過追蹤每個訪客瀏覽您網站的次數評估訪客的忠誠度。您可以檢視在選取時段內大多數瀏覽是來自首次還是第 20 次存取您網站的訪客。

造訪次數是以訪客的期限為準，而非所選的日期範圍。
