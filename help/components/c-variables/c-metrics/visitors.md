# 訪客

您的網站在選取的小時、日、週、月、季或年內的獨特訪客數量。

| 使用 | 說明 |
|---|---|
| 流量 | 訪客是指進入您的網站的人。不要求使用永久性 cookie。 |
| 轉換 | 訪客是指進入您的網站的人。當發生有關轉換的事件或動作時就會被計算為一次。 |
| Ad hoc analysis | 訪客是指進入您的網站的人。不要求使用永久性 cookie。 |

>[!NOTE] 如果您將 Analytics 當做 Adobe 的報表來源，請參閱 [!DNL Target] 文件中的[減少 A4T 中誇大的造訪和訪客計數](https://docs.adobe.com/content/help/en/target/using/integrate/a4t/troubleshoot-a4t/minimizing-inflated-visit-and-visitor-counts-a4t.html)。

如需詳細資訊，請參閱「Adobe Analytics 實作指南」中的[識別獨特訪客](https://docs.adobe.com/content/help/zh-Hant/analytics/technotes/visitor-identification.html)。
