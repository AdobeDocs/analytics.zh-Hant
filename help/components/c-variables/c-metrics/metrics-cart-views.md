# 購物車檢視

客戶檢視購物車內容的事件。此值來自於 scView 事件。

| 使用 | 說明 |
|---|---|
| 流量 | 不適用 |
| 轉換 | 設定[!UICONTROL 購物車檢視]事件的例項。發生於檢視購物車中的項目時。 |
| Ad Hoc Analysis | 設定[!UICONTROL 購物車檢視]事件的例項。發生於檢視購物車中的項目時。 |
