# 計算量度

您可以透過計算量度合併量度，以建立用作新量度的數學運算。您可以為您新增量度的報告建立這些量度。管理員可為報表套裝的所有使用者建立計算量度。

如需詳細瞭解 Adobe Analytics 中計算量度的建立與管理方式有何變動，請參閱[計算量度文件](/help/components/c-calcmetrics/cm-overview.md)。
