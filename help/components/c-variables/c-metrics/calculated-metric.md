# 計算量度

您可以透過計算量度合併量度，以建立用作新量度的數學運算。您可以為您新增量度的報告建立這些量度。管理員可為報表套裝的所有使用者建立計算量度。

如需 Adobe Analytics 中計算量度建立與管理方式變更的詳細資訊，請參閱[計算量度指南](https://marketing.adobe.com/resources/help/en_US/analytics/calcmetrics/)。
