# 彈回數

包含單一伺服器呼叫的瀏覽。例如，如果訪客未以傳送資料至Adobe的方式與頁面互動（例如按一下連結或視訊開始），則單頁瀏覽是彈回數。 如果一次瀏覽收到不只一個點擊，則不計算彈回數。
