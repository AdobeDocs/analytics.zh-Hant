# 彈回數

包含單一伺服器呼叫的瀏覽。 例如，如果訪客未以傳送資料至Adobe的方式與頁面互動（例如按一下連結或視訊開始），則單頁瀏覽是彈回數。 如果在一次瀏覽中收到多個點擊，則不會計算反彈。
