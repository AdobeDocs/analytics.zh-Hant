# 彈回數

包含單一伺服器呼叫的瀏覽。例如，如果訪客沒有用將資料傳送至 Adobe 的方式與頁面互動，例如按一下連結或啟動影片，則單次頁面瀏覽即為彈回數。如果一次瀏覽收到不只一個點擊，則不計算彈回數。
