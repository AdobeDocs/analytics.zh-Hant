# 單次存取

「單次存取」的定義為對您網站的存取包含單一獨特「頁面名稱」值的瀏覽次數。若使用者來到您的網站並點按了追蹤連結、觸發事件 (例如影片檢視) 或重新載入頁面，該次瀏覽仍會計為單次存取瀏覽。只要 pageName 變數的值未變更，不管傳送多少請求，該次存取仍計為「單次存取」。
