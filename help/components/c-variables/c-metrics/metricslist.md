# 量度概觀

量度是報告的基礎，有助於您檢視和理解資料關係，並可橫向比較您網站的不同資料集。量度是有關訪客活動的量化資訊，例如瀏覽次數、點進、重新載入、平均逗留時間、件數、訂購、收入等。

量度和相關資料顯示在報告欄中。流量量度會顯示訪客量的相關資料。轉換量度顯示成功事件 (例如購買、下載或任何其他您希望使用者在您網站上執行的動作) 的相關資料。
