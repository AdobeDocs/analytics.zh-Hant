# 購物車新增

「購物車新增」量度顯示訪客將物品新增至購物車的次數。 當您想要瞭解客戶對產品感興趣以將其新增至購物車的轉換漏斗部分時，此量度會很有幫助。

## 此度量的計算方式

此度量會計算變數中存 `scAdd` 在的點擊 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 數。
