# 人物

「人員」量度會根據Adobe的裝置圖表計算人員或裝置群組。 它更精確地表示獨特訪 [客數](unique-visitors.md)。

See the [People metric](https://docs.adobe.com/content/help/zh-Hant/device-co-op/using/data/people.html) in the Device Co-op user guide.
