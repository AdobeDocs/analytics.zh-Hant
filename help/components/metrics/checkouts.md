# 結帳

「結帳」量度會顯示訪客開始進行結帳程序的次數。如果您想要瞭解客戶已在購物車中放入產品並準備購買的轉換漏斗部分，此量度將有所幫助。

## 此量度的計算方式

此量度會計算 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 變數中有 `scCheckout` 存在的點擊次數。
