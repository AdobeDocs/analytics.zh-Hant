# 結帳

「結帳」量度顯示訪客啟動結帳程式的次數。 當您想要瞭解客戶擁有購物車產品並準備購買的轉換漏斗部分時，此量度會很有幫助。

## 此度量的計算方式

此度量會計算變數中存 `scCheckout` 在的點擊 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 數。
