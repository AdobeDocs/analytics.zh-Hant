# 串流媒體服務播放器狀態追蹤量度

串流媒體服務播放器狀態追蹤量度，提供透過串流媒體收集程式庫收集之資料的補充報告功能。 這些量度需要&#x200B;**[!UICONTROL 適用於串流媒體的Adobe Analytics附加元件]**。 如需詳細資訊，請聯絡您的Adobe客戶團隊。

若要使用這些量度，請為報表套裝啟用[[!UICONTROL 媒體報告]](/help/admin/tools/manage-rs/edit-settings/media-management.md)下的&#x200B;**[!UICONTROL 播放器狀態追蹤]**。

可使用下列量度：

* [[!UICONTROL 受隱藏式字幕影響的資料流]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/closed-captioning-streams-impacted)
* [[!UICONTROL 隱藏式字幕計數]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/closed-captioning-count)
* [[!UICONTROL 隱藏式字幕總時間]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/closed-captioning-total-duration)
* [[!UICONTROL 受全熒幕影響的資料流]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/full-screen-streams-impacted)
* [[!UICONTROL 全熒幕次數]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/full-screen-count)
* [[!UICONTROL 全熒幕總時間]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/full-screen-total-duration)
* [[!UICONTROL 受觀看中影響的資料流]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/in-focus-streams-impacted)
* [[!UICONTROL 觀看中次數]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/in-focus-count)
* [[!UICONTROL 觀看中總時間]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/in-focus-total-duration)
* [[!UICONTROL 受靜音影響的資料流]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/mute-streams-impacted)
* [[!UICONTROL 靜音計數]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/mute-count)
* [[!UICONTROL 靜音總時間]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/mute-total-duration)
* [[!UICONTROL 受子母畫面影響的資料流]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/picture-in-picture-streams-impacted)
* [[!UICONTROL 子母畫面計數]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/picture-in-picture-count)
* [[!UICONTROL 子母畫面總時間]](https://experienceleague.adobe.com/en/docs/media-analytics/using/reporting/metrics/picture-in-picture-total-duration)
