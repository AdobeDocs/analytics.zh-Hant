# 反彈率

「反彈率」量度顯示包含一個點擊的瀏覽次數與該頁面上的瀏覽次數之比。 您可以使用此度量來瞭解哪些維度值具有最高的反彈率，或查看網站隨時間的總反彈率。

## 此度量的計算方式

此度量是使用公式計算 [`Bounces`](bounces.md)` divided by `[`Entries`](entries.md)。
