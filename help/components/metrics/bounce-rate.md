# 跳出率

「反彈率」量度會顯示包含單一點擊的造訪次數與該頁面上的造訪次數之間的比率。您可以使用此量度來了解哪些維度項目具有最高的跳出率，或查看網站在一段時間內彙總的跳出率。高跳出率可說明網站或應用程式的問題，例如：設計不良、載入時間緩慢、不相關內容，或訪客意圖與頁面內容不符。

## 此量度的計算方式

此量度使用公式計算 [`Bounces`](bounces.md) `divided by` [`Entries`](entries.md).
