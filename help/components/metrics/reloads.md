# 重新載入

「重新載入」量度顯示維度項目在重新載入期間的顯示次數。 重新整理瀏覽器的訪客是觸發重新載入的最常見方式。

## 此量度的計算方式

此度量會計算維度包含與上次 [`Page`](../dimensions/page.md) 點擊相同值的點擊數。
