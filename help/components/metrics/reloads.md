# 重新載入

「重新載入」 [量度](overview.md) 顯示重新載入期間維度專案出現的次數。 訪客重新整理瀏覽器是觸發重新載入的最常見方式。

## 此量度的計算方式

此量度會計算 [`Page`](../dimensions/page.md) 維度包含與上次點擊相同值的點擊數。
