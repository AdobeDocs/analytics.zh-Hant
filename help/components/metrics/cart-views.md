# 購物車檢視

「購物車檢視」[量度](overview.md)會顯示訪客檢視其購物車的次數。 如果您想要瞭解客戶檢視其購物車內容的轉換漏斗部分，此量度將有所幫助。

## 此量度的計算方式

此量度會計算 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 變數中有 `scView` 存在的點擊次數。

如需量度比較，請參閱[購物車](carts.md)。
