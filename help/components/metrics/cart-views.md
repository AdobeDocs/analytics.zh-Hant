# 購物車檢視

「購物車檢視」量度顯示訪客檢視其購物車的次數。 當您想要瞭解客戶在購物車中檢視內容的轉換漏斗部分時，此量度會很有幫助。

## 此度量的計算方式

此度量會計算變數中存 `scView` 在的點擊 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 數。
