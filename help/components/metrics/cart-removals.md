# 購物車移除

「購物車移除」量度顯示訪客從購物車中移除某物的次數。 當您想要瞭解客戶不再對產品感興趣的轉換漏斗部分時，此度量會很有幫助。

## 此度量的計算方式

此度量會計算變數中存 `scRemove` 在的點擊 [`events`](/help/implement/vars/page-vars/events/events-overview.md) 數。
