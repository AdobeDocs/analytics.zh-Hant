# 各角色的區段權限

區段功能主要因管理員層級使用者和非管理員而異。

|  | 管理員使用者 | 非管理員 |
| --- | --- | --- |
| 套用區段 | 擁有的區段或共用的區段。 不過，您可以與自己共用任何使用者的區段 | 僅擁有的區段或共用區段 |
| 在區段管理器中檢視區段 | 組織內的所有區段 | 僅擁有的區段或共用區段 |
| 管理區段 | 可以檢視、編輯、刪除或核准任何使用者的區段 | 只能編輯或刪除自己的區段。 無法核准區段。 |
| 共用區段 | 可與整個公司、使用者群組或個人共用 | 只能與個人共用 |
