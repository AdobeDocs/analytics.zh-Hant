# 報表存取

說明如何執行行銷渠道報表、授予有限的管理員使用者權利以及報表的使用者群組權限。

若要檢視行銷渠道報表，您必須授予使用者群組適當的權限。Adobe Experience Cloud 的權限系統是以群組為基礎，換言之，使用者的群組決定了存取報表的權限。
