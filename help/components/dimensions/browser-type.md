# 瀏覽器類型

「瀏覽器類型」維度會列出建立訪客使用之瀏覽器的組織。 當您想要查看訪客使用的主要瀏覽器時，此維度很實用。 它提供「瀏覽器」維度的值，因為它不會將相同瀏覽器的不同版本列為個別的維度項目。

## 將資料填入此維度

此維度會參照Adobe內部的查閱表格。 查閱值是以影像請求中 `User-Agent` 的HTTP標題為基礎。 如果您使用AppMeasurement程式庫（例如透過Adobe Experience Platform Launch），此維度就可立即運作。

## 維度項目

維度項目包括製作瀏覽器的組織。 常見維項 `"Google"` 目(建立者 [!DNL Chrome])、 `"Apple"` (建立者 [!DNL Safari])、 `"Microsoft"` (建立者 [!DNL Edge]) `"Mozilla"`[!DNL Firefox]和（建立者）。
