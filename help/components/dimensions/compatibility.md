# Analytics 維度相容性

此頁面列出各自Analytics功能支援的維度。

>[!NOTE]
>
>自訂變數名稱、分類和訪客屬性會在此清單中忽略。 這些維度項目是個別報表套裝專屬的。

>[!NOTE]
>
>Analytics工具對類似維度使用不同術語時，有些重疊。 例如，「報告與分析」在「分析工作區」 `browserwidth` 使用時會使用 `browserwidthbucketed`。
