# 週

「周」維度會報告指定量度發生的周。 第一個維度項目是日期範圍內的第一週，最後一個維度項目是日期範圍中的最後一週。 此維度對於趨勢報表至關重要，因為它可讓您查看隨時間變化的量度。

## 將資料填入此維度

此維度適用於所有實作，不需開箱即用。 如果報表套裝包含資料，此維度會運作。

## 維度項目

維度項目包括一週中第一天的日期（月、日和年）。
