# 季

「季」維度會報告指定的量度發生的季度。第一個維度項目是日期範圍中的第一季，最後一個維度項目是日期範圍中的最後一季。此維度可讓您查看量度在一段時間內變化，因此非常適合用於趨勢報表。

## 將資料填入此維度中

此維度可直接用於所有實施作業。如果報表套裝包含資料，此維度即會運作。

## 維度項目

維度項目包含指定日期 3 個月的季度和年份。
