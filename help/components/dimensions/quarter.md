# 季度

「季」維度會報告指定量度發生的季。 第一個維度值是日期範圍中的第一季，最後一個維度值是日期範圍中的最後一個季。 此維度最適合趨勢報表，因為它可讓您查看隨時間變化的量度。

## 將資料填入此維度

此維度適用於所有實作，不需開箱即用。 如果報表套裝包含資料，此維度會運作。

## 維度值

維度值包括指定日期的3個月季度和年。
