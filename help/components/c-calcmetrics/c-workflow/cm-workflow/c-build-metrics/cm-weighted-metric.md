## 已篩選的反彈率 {#section_D42F2452E4464948934063EB6F2DAAB4}

這個簡單的已篩選量度只顯示超過 100 次瀏覽之頁面的反彈率:

![](assets/cm_fbr.png)

請記住，此公式相依於一致的時間範圍。如果您針對單日執行報表，則超過 20 次瀏覽的頁面都值得查看。如果您針對一個月執行報表，可能會想讓篩選包含更多次瀏覽。
